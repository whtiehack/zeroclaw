# `docs/master_wecom`

本目录只记录 `master_wecom` 分支专属的开发文档，不作为上游通用文档目录使用。

适合放在这里的内容：

- 本地定制需求说明
- `master_wecom` 分支专属的设计决策
- 个人工作流记录、操作备忘、回滚说明
- 不准备直接提交到上游仓库的分支侧文档

提交规则：

- 本目录下的改动必须单独提交。
- 不要把本目录文档和代码改动混在同一个 commit。
- 不要把本目录文档和准备提交到上游的通用文档混在同一个 commit。
- 需要给上游提 PR 时，默认不携带本目录提交；如确有必要，应先整理成独立的上游通用文档提交。

目标：

- 保持 `master_wecom` 分支的私有记录清晰可追踪。
- 保持可提交上游的 commit 干净、易 cherry-pick、易审查。
